# 12HoursFormatTo24Hours
Midnight is 12:00:00AM on a 12-hour clock, and 00:00:00 on a 24-hour clock. Noon is 12:00:00PM on a 12-hour clock, and 12:00:00 on a 24-hour clock. The above function will convert 12 hour Time format to 24 hours format.
